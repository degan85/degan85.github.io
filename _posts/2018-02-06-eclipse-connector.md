## Oxygen 이전

- subversive - SVN Team Provider install

- Connector 설치가 뜸

## Oxygen 이후
- restart 후 Connector 설치가 안뜸
- [Help - install new software] 에서 아래 주소를 입력
	- http://community.polarion.com/projects/subversive/download/eclipse/6.0/update-site
